## Functional Requirements

### Authentication & Authorization:
✓ Role-based access control (Super Admin, Owner, Executive Manager)
✓ Protected routes based on user roles
✓ User session management
✓ Login/logout functionality

### User Management:
✓ Super Admin can manage all farm owners
✓ Owner profile with CNIC and contact details
✓ Pakistani CNIC format validation (13 digits)
✓ Pakistani phone number validation (11 digits)

### Farm Management:
✓ Create, read, update, delete farms
✓ Farm statistics (total cows, active/inactive collars)
✓ Location-based farm organization
✓ Owner assignment to farms
✓ Multi-farm support for owners

### Animal Management:
✓ Track animals with collar IDs
✓ Monitor age, breed, weight
✓ Real-time temperature monitoring
✓ Activity level tracking (Low, Normal, High)
✓ Health status monitoring (Healthy, Unhealthy)
✓ Estrus cycle detection
✓ Add, update, remove animals

### Health Records:
✓ Create and manage health records
✓ Link records to collar IDs
✓ Veterinarian information tracking
✓ Treatment and notes documentation
✓ Date-stamped records
✓ Filtering and search capabilities

### Employee Management:
✓ Assign executive managers to farms
✓ One manager per farm validation
✓ Employee contact information
✓ Join date tracking
✓ Manager assignment/reassignment

### Dashboard Features:
✓ Role-specific dashboards
✓ Real-time statistics and charts
✓ Quick action cards
✓ Data visualization with recharts
✓ Responsive layout

### Data Validation:
✓ Name validation (no numbers allowed)
✓ Numeric field validation (no negative numbers)
✓ CNIC format: XXXXX-XXXXXXX-X
✓ Phone format: 11 digits (Pakistani)
✓ Email validation
✓ Required field validation
✓ Real-time input filtering

### UI/UX Requirements:
✓ Responsive design (mobile, tablet, desktop)
✓ Glassmorphic card design
✓ Modal popups for all CRUD operations
✓ Dark/light theme support
✓ Loading states and animations
✓ Toast notifications for user feedback
✓ Color-coded status badges
✓ Filterable data tables
✓ Confirmation dialogs for delete actions

## Installation Steps:

1. Install Node.js (v18+)
2. Install pnpm: npm install -g pnpm
3. Clone the repository
4. Run: pnpm install
5. Run: pnpm dev
6. Access: http://localhost:3000

## Environment:
- Development: pnpm dev (localhost:3000)
- Production: pnpm build && pnpm start
- Linting: pnpm lint
